# Hackclub Game



### 1. Create a Virtual Environment

``` bash
python -m venv .venv
```

### 2. Activate the Virtual Environment

**On Linux / FreeBSD:**

``` bash
source .venv/bin/activate
```

### 3. Install Dependencies

``` bash
pip install pygame
```

### 4. Run the Game

``` bash
python main.py
```

Enjoy playing!
